# CRUD Flask & Vue

## Instrucciones

### Download Project
`https://github.com/Gizmo0oo/flask-vue-crud`

`cd flask-vue-crud`

### Run server

#### Windows
`cd server`
`env\Scripts\activate.bat`
`python app`

#### Linux
`cd server`
`source env/bin/activate`
`python app.py`

### Run client
`cd client`
`npm run serve`

### Open your browser
Open youre browser on `localhost:5000`
